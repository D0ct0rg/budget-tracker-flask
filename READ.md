# Budget Tracker

A Flask-based web application for tracking income and expenses using SQLite.

## Features

### Current Features

* SQLite database integration
* CRUD operations for transactions
* Summary statistics

  * Current Balance
  * Total Income
  * Total Expenses
* Transaction table display
* Flask backend with Jinja templates

### Planned Features

* Add transaction form
* Edit transactions
* Delete transactions
* Category filtering
* Dark mode
* Improved styling

## Technologies Used

* Python
* Flask
* SQLite
* HTML/CSS
* Jinja2
